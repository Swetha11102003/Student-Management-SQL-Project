# Student Management SQL Project
  - A beginner-friendly SQL data analysis project for student performance analysis

# Objective
  - Analyze student performance and generate business insights using SQL.

# Skills Used
  - SELECT
  - WHERE
  - GROUP BY
  - HAVING
  - INNER JOIN
  - CASE
  - Subqueries
  - CTE
  - Window Functions

# Database Schema
  - Students
  - Subjects
  - Marks
    
# Business Questions
  - Overall topper
  - Subject-wise average
  - Top 3 students
  - Pass/Fail analysis
  - Above-average performers

# Business Insights
- Identified the overall topper.

- Found the top 3 performing students.
- Identified above-average performers.
- Analyzed pass/fail status.
